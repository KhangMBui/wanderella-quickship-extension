Privacy Policy — Wanderella QuickShip

Last updated: January 18, 2026

Wanderella QuickShip is a Chrome extension that streamlines the shipping workflow by allowing a user to copy destination and order details from Wanderella (WooCommerce admin) and paste them into LetterTrackPro with a button click.

Data the extension accesses

When you click the extension’s copy/paste buttons, the extension may access order-related fields that appear on the page, such as:

  • Recipient name
  • Company (if present)
  • Phone number (if present)
  • Destination address (street, city, state, postal code, country)
  • Email address (if present)
  • Order number

How the extension uses data

The extension uses the above data only for its single purpose: transferring destination/order details from Wanderella to the matching destination fields in LetterTrackPro at your request.

Storage

The extension stores copied destination/order data locally in your browser using chrome.storage.local so it can be pasted on LetterTrackPro. This data stays on your device unless you clear it (for example by clearing extension data or uninstalling the extension).

Data sharing and selling

The extension does not sell user data. The extension does not transmit data to external servers or share data with third parties.

Analytics and tracking

The extension does not collect analytics, does not track browsing activity, and does not log keystrokes or user behavior.

Security

The extension performs all processing locally in the browser. Because copied data may include personally identifiable information (PII), you should only use the extension on trusted devices and accounts.

Changes to this policy

If the extension’s data practices change, this privacy policy will be updated to reflect the new practices.
